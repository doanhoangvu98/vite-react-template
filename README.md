# Instructions

## Install package

`npm install`

## 1. Check ESlint error

`npm run lint`

## 2. Preview

`npm run preview`

## 3. Auto fix ESlint problem

`npm run lint:fix`

## 4. Check prettier error

`npm run prettier`

## 5. Auto fix Prettier error

`npm run prettier:fix`

## 6. Run test

`npm run test`

## 7. Run project

`npm run dev`
